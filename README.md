# blocktech.nl

> Landing page and course website for block tech @cmda.

## Description

Overview of all the courses in block-tech with learning goals and instructors. For upcoming students but also explaining this course for external guest speakers and those who seek additional information.

## Install

Follow these steps to get started:

1. Clone the project

```
$ git clone https://github.com/cmda-bt/blocktech.nl.git
```

2. Then do the

```
npm install
```

3. And then the

```
npm run dev
```

to start the live-reloadzz on a local development server.
